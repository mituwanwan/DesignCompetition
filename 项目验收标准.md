# 智慧养老院管理系统 - 项目验收标准

## 1. 项目概述

### 1.1 项目背景
智慧养老院管理系统是一个完整的养老院信息化解决方案，包含管理员、护工、家属三类角色，实现了老人信息管理、健康数据采集、报警中心、探视预约、消息沟通等核心功能。

### 1.2 技术栈
- 后端: Python Flask
- 数据库: SQLite
- 前端: HTML5 + Bootstrap5 + JavaScript
- 可视化: Plotly
- 文档导出: pandas + openpyxl

## 3. 非功能验收标准

### 3.1 性能要求
- 页面首次加载时间 ≤ 3秒
- API响应时间 ≤ 2秒
- 数据库查询优化，避免全表扫描
- 支持并发用户至少 50人

### 3.2 安全要求
- 密码哈希存储（SHA256+Salt）
- 登录失败锁定机制
- Session过期保护
- 基于角色的权限控制
- XSS防护（模板自动转义）
- CSRF防护（Token验证）

### 3.3 兼容性要求
- 支持现代主流浏览器（Chrome、Firefox、Safari、Edge）
- 响应式设计，支持PC端和平板设备
- 跨平台运行（Windows、Linux、Mac）

### 3.4 用户体验
- 界面统一，采用Bootstrap5风格
- 所有操作有明确反馈
- 错误提示友好易懂
- 响应式布局，自适应不同屏幕
